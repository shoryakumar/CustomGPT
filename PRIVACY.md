# Privacy Policy for Biorhyme Health Chatbot

**Last Updated: November 16, 2025**

## Overview

This is a **demonstration/proof-of-concept** health chatbot created for educational and testing purposes. This privacy policy explains how data is collected and used.

## Important Notice

⚠️ **THIS IS A DEMO APPLICATION**

- Do NOT enter real personal health information
- Do NOT use for actual medical decisions
- This is for demonstration purposes only
- Not HIPAA compliant

## Data Collection

### What We Collect

When you interact with this chatbot, we collect:

1. **Chat Messages**: Questions and responses exchanged with the chatbot
2. **Meal Data**: Meal names, nutrition information, and timestamps
3. **Medication Data**: Medication names, dosages, and frequencies (for demo purposes)
4. **Usage Data**: Timestamps of interactions

### What We DON'T Collect

- Real names or personal identifiers
- Email addresses
- Payment information
- Real medical records
- Location data

## How We Use Data

Data collected is used solely for:

1. **Demonstration purposes** - To show chatbot functionality
2. **Testing** - To verify the system works correctly
3. **Development** - To improve the demo application

## Data Storage

- All data is stored in a demo database
- Uses shared demo user account
- No separation of individual user data
- May be reset or deleted at any time

## Data Sharing

We do NOT:

- Sell your data
- Share data with third parties
- Use data for marketing
- Store data long-term

## Your Rights

Since this is a demo application:

- All data can be reset using the `/api/reset/` endpoint
- No persistent user accounts
- No data export functionality (demo only)

## Security

- HTTPS encryption for data in transit
- No authentication required (demo mode)
- Not suitable for real health data

## Third-Party Services

This application may interact with:

- **OpenAI/ChatGPT** - For AI-powered responses
- **AWS Services** - For hosting and database

These services have their own privacy policies.

## Medical Disclaimer

This chatbot is NOT:

- A medical device
- A substitute for professional medical advice
- Intended for diagnosis or treatment
- HIPAA compliant

Always consult qualified healthcare professionals for medical decisions.

## Children's Privacy

This demo is not intended for use by children under 13.
